{% trans "What's next?" %}

{% trans "We know it is not easy to find alternatives in order to escape trackers within our apps. We prepared a list of tools and things you can do to better manage your privacy on your smartphone." %}

{% trans "Consider how you use applications" %}

{% trans "When you wish to install an application or when an application asks for personal information, ask yourself the following questions:" %}

  • {% trans "how many trackers does it contain and what permissions does it ask for?" %}
  • {% trans "do I want or need to install it?" %}
  • {% trans "do I want or need to give it what it asks me?" %}

{% trans "It is up to you to decide but by taking into consideration your daily use, you can have a better control over your digital identity." %}

{% trans "You could also contact the people who offer you the application if you consider it to be too intrusive and ask them some explanations on the presence of trackers and permission requests." %}

{% trans "Favour free software" %}

{% trans "What is called 'free' software is software whose code (i.e. its recipe, or list of instructions) can be changed, copied, verified... and is thus transparent. A free software application might exist to replace an app that you already use but that may not respect your privacy." %}

{% trans "If you are an Android user, you can install F-Droid. It is a free software store that can be used instead or in addition to the Google Play store. Do not hesitate to refer to the F-Droid documentation if needed. Be aware, however, that some F-Droid applications are not available in all languages." %}

{% trans "Use the mobile website instead of the application" %}

{% trans "On a mobile device you don't necessary have to use applications: you can use mobile websites instead. One advantage in favour of the latter is the possibility to use web browser add-ons, in the same way you can do it on your computer (e.g. uBlock Origin)." %}

{% trans "If you want a browser experience that is more respectful of your privacy, you can use alternative browsers such as Mozilla Firefox or the DuckDuckGo Privacy Browser." %}

{% trans "These browsers allow you to add your favorite websites on your home screen easily, just as an app (but lighter)." %}

{% trans "Install an ad blocker" %}

{% trans "In all cases, Android smartphones allow you to install ad-blocking applications such as Blokada. You can install it from F-Droid (cf. above) or from their own website." %}
